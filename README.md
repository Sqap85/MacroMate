## English

MacroMate is a modern web application for daily macro tracking and nutrition management. Users can add foods, set macro goals, view past records, make plans, and easily track their statistics.

### Features
- User authentication & email verification
- Add foods and track macro values
- Set goals and view statistics cards
- View history of entries and future planning
- Profile management
- Quick add with food templates
- Mobile compatibility (responsive)

### Technologies Used
- React
- TypeScript
- Vite
- Firebase
- Formik
- Yup
- MUI (Material UI)
- CSS
